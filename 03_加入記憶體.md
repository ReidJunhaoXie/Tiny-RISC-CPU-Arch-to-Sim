# 03_加入記憶體
- 製作一個簡易 RAM 

在處理器架構中，**記憶體（Memory）** 被用來存放資料以及程式。這裡用 Register 實現簡單的「多位址儲存單元」，並說明基本讀寫原理，這進一步實現資料的自動載入。

---

## 1. Register Array 記憶體

- **設計思路：**
    - 將 8 個 Register 並聯，相當於 8 筆可獨立控制的資料儲存格。
    - 每個 Register 都有唯一的位址。
    - 以 3 bits address 配合 **3-to-8 Decoder** 解碼，將對應的 Register enable 拉高。

## 2. 讀(load)/寫(store)控制機制

### 2.1 寫入（str）

- 寫入時機：
    - 當 `str`（store）訊號有效，且經由 Decoder 使對應 Register 的 `en`（enable）有效。
    - `str` 和 Register `en` 做 AND 後，啟動寫入。
- 也就是str加在RAM的輸入端
- str控制示意：

### 2.2 讀出（ld）

- 讀出時機：
    - 當 `ld`（load）訊號有效，使用 **tri-state buffer** 控制資料是否送到輸出端。
    - Decoder 指向對應 Register，使其輸出端可被選通到 輸出。
    - 也就是ld控制加在RAM輸出端
    - 加個下拉電阻避免FLOATING
